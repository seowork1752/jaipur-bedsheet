🚀 HOW TO GET STARTED (Copy-Paste):

1️⃣  SETUP (5 MINUTES)
━━━━━━━━━━━━━━━━━━━━

# Navigate to project
cd jaipur-bedsheets-ecommerce

# Install dependencies
npm run install-all

# Copy environment file
cp backend/.env.example backend/.env

# Edit backend/.env with:
#   MONGODB_URI = your_mongodb_connection
#   RAZORPAY_KEY_ID = your_razorpay_key
#   RAZORPAY_KEY_SECRET = your_razorpay_secret

# Start development servers
npm run dev

# Visit:
#   Frontend: http://localhost:3000
#   Backend API: http://localhost:5000


2️⃣  CONFIGURE SERVICES

MongoDB Atlas (Free):
  1. Go to mongodb.com/cloud/atlas
  2. Create free cluster
  3. Create database user
  4. Get connection string
  5. Add to backend/.env

Razorpay (Free Test Account):
  1. Go to razorpay.com
  2. Create account
  3. Get test API keys from dashboard
  4. Add to backend/.env


3️⃣  POPULATE DEMO DATA

cd backend
npx ts-node src/scripts/seedData.ts

(Creates 25+ sample products for testing)


4️⃣  TEST THE PLATFORM

Frontend:
  - Visit http://localhost:3000
  - Browse products
  - Add to cart
  - Login (any test account)
  - Proceed to checkout

Payments (Test Credentials):
  - UPI: success@razorpay
  - Card: 4111 1111 1111 1111 (12/25, 123)
  - Coupon: WELCOME10 (10% off)
